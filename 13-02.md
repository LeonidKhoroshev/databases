# Домашнее задание к занятию  «Защита хоста» - Леонид Хорошев

------

### Задание 1

1. Установите **eCryptfs**.
Домашнее задание выполняется на ВМ Alma linux, развернутой в Яндекс Облаке. Утилита eCryptfs отсутствует в исходном репозитории, пожтому сначала добавляем необходимый репозиторий, а затем устанавливаем eCryptfs.
```
wget http://mirror.rackspace.com/elrepo/elrepo/el8/x86_64/RPMS/elrepo-release-8.3-1.el8.elrepo.noarch.rpm
rpm -Uvh elrepo-release-8.3-1.el8.elrepo.noarch.rpm
yum install ecryptfs-utils
```
2. Добавьте пользователя cryptouser.
3. Зашифруйте домашний каталог пользователя с помощью eCryptfs.


*В качестве ответа  пришлите снимки экрана домашнего каталога пользователя с исходными и зашифрованными данными.*
